Online and student made tools

Cascade

Cascade is a note taking tool made by Phoenix,
one of this websites co-creators.
please note that it is still in development

To use, either download the zip, extract and run the html file, or use the links in the description of the repository

Tor Browser

Tor is a way of getting past the schools wifi restricitons,
but it puts you at a slower speed

Follow the download instructions on the website

OCDSB guest login

If you don't want the school to track your information,
the schools guest wifi is the better option

To use, look for it in wifi settings while at school

Windscribe

Out of all vpns, windscribe is pretty ok.
At the free level, you always get the best speed you can get, with a 10gb a month limit.
More than enough for browsing, but a little limiting for playing online games

Follow download instructions on the official website
