Note-taking and Organization tools

Cascade

Cascade is a note taking tool made by Phoenix,
one of this websites co-creators.
please note that it is still in development

To use, either download the zip, extract and run the html file, or use the links in the description of the repository

OneNote

OneNote is a great simple tool made by microsoft for writing small documents and taking notes

Follow the download instructions on the website to try this tool

Trello

Trello is a great web based tool that can be used to organize the tasks you need to complete, for a project, or just to make a list of tasks that need to be done

You can get started with Trello by following the instructions on its website

Online safety Resources

Tor Browser

Tor is a great way to protect yourself online, obscuring your data from dangerous people on the web,
but it puts you at a slower speed

Follow the download instructions on the website if you want to keep your browsing data to yourself when browsing online

Windscribe

Windscribe is similar to tor, however, it faster, but less secure. The main drawback is that the free version limits you with a 10gb a month limit.

Follow download instructions on the official website to start protecting yourself online

Firefox

Firefox is a beautiful, clean, browser that respects your privacy. Unlike other browsers, which are all based on chrome, firefox is built from the ground up, and asks for permission to use your data, and lets you stop them from using your browsing data at any time.

Follow the instructions on Mozilla's website to start using Firefox
